실제와 그에 대한 주장을 이진법으로 분류해보면 다음과 같은 네가지 경우가 발생한다.

|     |              |       실제       |                |
| :-: | :----------: | :------------: | :------------: |
|     |              |    실제가 참인경우    |   실제가 거짓인 경우   |
| 주장  | 참이라 주장하는 경우  | True Positive  | False Positive |
|     | 거짓이라 주장하는 경우 | False Negative | True Negative  |
암진단의 예를 들어 생각해 보면 이해가 쉬울 것이다. 진실에 대한 주장도 위와 같은 결과들을 갖는다.
